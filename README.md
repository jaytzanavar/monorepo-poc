## Start the app 

To start the development server run `nx serve tour`. Open your browser and navigate to http://localhost:4200/. Happy coding!

## APP Description

This is a Client Admin basic portal splitted in different libraries.
Main app is tour.
Secondary libraries that access the Points of interest. Visitor (visitor Lib)
Admin (Admin Lib with statistics)
And A feature state in POI lib
## Ready to deploy?

Just run `nx build demoapp` to build the application. The build artifacts will be stored in the `dist/` directory, ready to be deployed.
